## Tier Strategy

InvestorClaw uses an **open-core + free tier** model:

### Free Tier (Apache 2.0 Dual License)

Core portfolio analysis features available to everyone:

- **Holdings Analysis** — View holdings, prices, allocation
- **Performance Tracking** — Returns analysis, metrics, heat maps
- **News Sentiment** — Fetch news, sentiment analysis
- **Analyst Ratings** — Basic analyst recommendation data (market data providers)
- **Portfolio Export** — CSV/Excel export
- **Self-Hosted** — Run on Raspberry Pi (2GB+) with zeroclaw
- **Educational Guardrails** — Single-investor mode with compliance checks

### Premium Features (Proprietary, Paid)

Advanced capabilities for professional investors and advisors:

- **Multi-Portfolio Management** — Manage multiple investor accounts
- **Tax-Loss Harvesting** — Strategic tax optimization
- **ETF Expansion** — Deep constituent analysis
- **Advisor Mode** — Professional FA features with full guardrails
- **Cloud Hosting** — Managed cloud deployment + API access
- **Compliance & Audit** — 7-year audit trail for fiduciary advisors
- **Real-Time Updates** — WebSocket + streaming market data
- **Advanced Consultation** — Tier-3 enrichment with extended context

## Deployment

- **Free tier:** Self-hosted (Mac with OpenClaw or Pi with zeroclaw)
- **Premium tier:** Cloud-hosted managed service (coming 2026)
